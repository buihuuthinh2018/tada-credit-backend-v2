# Project Overview

This is a financial service & credit management system.

Main features:
- User registration with OTP optional (admin configurable)
- Dynamic role-based access control (RBAC)
- Referral & commission ecosystem
- Config-driven services, documents, workflows
- Financial wallet with ledger accuracy
- Admin-controlled approval flows
- Full audit logging

System must be scalable, auditable, and business-configurable.
